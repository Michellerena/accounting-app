# 极简记账应用 📊

一个功能完整的个人记账 Web 应用，支持用户注册登录、收支记录、数据统计和日期筛选。

## ✨ 功能特性

- 🔐 用户注册与登录（密码加密）
- 💰 添加/编辑/删除收支记录
- 📊 实时统计总收入、总支出和余额
- 📅 按日期范围筛选记录
- 📱 响应式设计，支持手机和电脑
- 🗄️ 数据持久化存储
